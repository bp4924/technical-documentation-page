FizzBuzz

The following JavaScript code will recreate the party game FizzBuzz.

Rules of FizzBuzz:

  1. Participants sit in a circle.
  2. The game begins with one person saying "one."
  3. The next person in the circle says the next number in sequence, but with the following rules:
    • If the number is divisible by three, they say "Fizz"
    • If the number is divisible by five, they say "Buzz"
    • If the number is divisible by three and five, they say "FizzBuzz"
    • Otherwise, they say the number
  4. If a participant makes a mistake, they are out of the came.
  5. The game continues until only one person is left. That person is declared the winner!!

FizzBuzz solution

FizzBuzz is often used as a programming exercise to practice conditional elements and loops.

Solution using JavaScript:

          
            const fizzBuzz = 45; 

            for (let i = 1; i = fizzBuzz; i++) { 
              if (i % 5 === 0 && i % 3 === 0) {
              console.log(i + " FizzBuzz"); 
              } else if (i % 5 === 0) { 
                console.log(i + " buzz"); 
              } else if (i % 3 === 0 ? console.log(i + " fizz") : console.log(i));
            }
